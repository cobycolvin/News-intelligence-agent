# Repository Agent Instructions

## Scope
These instructions apply to the full repository.

## Conventions
- Keep backend code modular under `backend/app`.
- Keep frontend code modular under `frontend/src/components` and `frontend/src/api`.
- Maintain mock-mode compatibility for all core features.
- Add tests when adding or changing behavior.
- Keep documentation synchronized with implementation.

## Validation
- Run backend tests with `pytest` from `backend/`.
- Run frontend tests with `npm test` from `frontend/`.
- Build frontend with `npm run build`.
